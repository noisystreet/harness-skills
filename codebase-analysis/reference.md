# Codebase Analysis References

## Guides

- [C4 Model](https://c4model.com/) - communicating structure at multiple zoom levels.
- `Code Reading` / architecture recovery practices - evidence-based reading.
- Pair with `software-architecture` references for style vocabulary.

## What To Learn

- Start from entrypoints, build/test commands, and one primary flow.
- Separate facts from inferences in briefings.
- Use tests and types as oracles for intended behavior.

## Caveats

- Diagrams without file evidence drift quickly; keep links to symbols/paths.
