## Install

```bash
git clone https://github.com/noisystreet/harness-skills.git
cd harness-skills
make install            # Cursor-compatible: ~/.cursor/skills
# make install-trae     # Trae international: ~/.trae/skills
# make install-trae-cn  # Trae CN: ~/.trae-cn/skills
make check
make help
```

Project-local use: link or copy needed skill directories into your AI tool's skills/rules folder. See the repository `README.md` for Claude Code / Codex / Copilot usage.
