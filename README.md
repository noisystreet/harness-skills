| еҶҷ/ж”№ж–ҮжЎЈ | `docs-style` |
| ж–°йЎ№зӣ®еҲқе§ӢеҢ– | `project-bootstrap` + `docs-style` + `ci-quality` + еҜ№еә”иҜӯиЁҖ `*-style` |
| дёҚзЎ®е®ҡиҜҘз”Ёд»Җд№Ҳ | `development-workflow` |

### еёёз”Ёи§ҰеҸ‘иҜӯ

- гҖҢжҢү `code-review` е®ЎжҹҘиҝҷж¬Ўж”№еҠЁгҖҚ
- гҖҢжҢү `api-design` и®ҫи®ЎиҝҷдёӘжҺҘеҸЈгҖҚ
- гҖҢжҢү `data-modeling` жўізҗҶдёҚеҸҳйҮҸе’Ңиҫ№з•ҢгҖҚ
- гҖҢжҢү `migration` еҒҡиҝҷж¬Ў schema иҝҒз§»гҖҚ
- гҖҢжҢү `release` еҲҮ v0.1.0 е№¶еҸ‘зүҲгҖҚ
- гҖҢжҢү `python-style` еҶҷиҝҷдёӘжЁЎеқ—гҖҚ
- гҖҢжҢү `debugging` её®жҲ‘жҺ’жҹҘиҝҷдёӘеӨұиҙҘгҖҚ
- гҖҢжҢү `codebase-analysis` еҲҶжһҗиҝҷдёӘд»“еә“/еә“зҡ„жәҗз ҒгҖҚ
- гҖҢжҢү `software-architecture` иҜ„дј°жЁЎеқ—иҫ№з•Ңе’Ңжһ¶жһ„йЈҺж јгҖҚ
- гҖҢжҢү `performance` дјҳеҢ–иҝҷжқЎж…ўи·Ҝеҫ„гҖҚ
- гҖҢжҢү `refactoring` е®үе…ЁйҮҚжһ„иҝҷдёӘжЁЎеқ—гҖҚ
- гҖҢжҢү `observability` иЎҘйҪҗж—Ҙеҝ—е’ҢжҢҮж ҮгҖҚ
- гҖҢжҢү `dependency-management` е®ЎжҹҘиҝҷж¬Ўдҫқиө–еҚҮзә§гҖҚ
- гҖҢжҢү `development-workflow` е®һзҺ°иҝҷдёӘеҠҹиғҪгҖҚ
- гҖҢжҢү `docs-style` жӣҙж–° README е’Ң ADRгҖҚ
- гҖҢжҢү `runtime-reliability` жЈҖжҹҘиҝҷдёӘ workerгҖҚ
- гҖҢжҢү `testing` з»ҷиҝҷдёӘ bug иЎҘеӣһеҪ’жөӢиҜ•гҖҚ
- гҖҢжҢү `github-flow` ејҖ PRгҖҚ
- гҖҢжҢү `ci-quality` й…ҚзҪ® CI е’Ңжң¬ең°жЈҖжҹҘгҖҚ
- гҖҢжҢү `project-bootstrap` еҲқе§ӢеҢ–дёҖдёӘ Rust/Python/C++ йЎ№зӣ®гҖҚ

### йҖҡз”Ёз”Ёжі•

иҝҷдәӣ skill жң¬иҙЁдёҠжҳҜ Markdown жҢҮд»Өж–Үд»¶гҖӮдёҚеҗҢ AI зј–зЁӢе·Ҙе…·еҸҜд»Ҙз”ЁдёҚеҗҢж–№ејҸеј•з”Ёпјҡ

| е·Ҙе…·зұ»еһӢ | дҪҝз”Ёж–№ејҸ |
|----------|----------|
| ж”ҜжҢҒ Skills зӣ®еҪ•зҡ„е·Ҙе…· | е°ҶйңҖиҰҒзҡ„ skill зӣ®еҪ•еӨҚеҲ¶жҲ–з¬ҰеҸ·й“ҫжҺҘеҲ°иҜҘе·Ҙе…·зҡ„ skills зӣ®еҪ• |
| ж”ҜжҢҒ `AGENTS.md` / `CLAUDE.md` / йЎ№зӣ®и§„еҲҷзҡ„е·Ҙе…· | еңЁйЎ№зӣ®и§„еҲҷж–Үд»¶йҮҢеј•з”Ёжң¬д»“еә“еҜ№еә” `SKILL.md`пјҢжҲ–еӨҚеҲ¶зІҫз®Җи§„еҲҷ |
| Chat/CLI зұ»е·Ҙе…· | еңЁд»»еҠЎдёӯжҳҺзЎ®иҜҙгҖҢжҢү `<skill-name>`гҖҚпјҢе№¶йҷ„дёҠжҲ–и®©е·Ҙе…·иҜ»еҸ–еҜ№еә” `SKILL.md` |
| дёҚж”ҜжҢҒиҮӘеҠЁеҸ‘зҺ°зҡ„е·Ҙе…· | жүӢеҠЁжҠҠзӣёе…і `SKILL.md` еҶ…е®№дҪңдёәдёҠдёӢж–ҮжҸҗдҫӣ |

жҺЁиҚҗе…ҲдҪҝз”Ё `development-workflow` еҒҡи·Ҝз”ұпјҢеҶҚжҢүд»»еҠЎиҜ»еҸ–е…·дҪ“ skillгҖӮ

### йЎ№зӣ®еҶ…еј•з”Ё

жҠҠйңҖиҰҒзҡ„ skill й“ҫеҲ°жҲ–еӨҚеҲ¶еҲ°зӣ®ж ҮйЎ№зӣ®зҡ„ AI е·Ҙе…·и§„еҲҷзӣ®еҪ•гҖӮзӨәдҫӢпјҡ

```bash
# дҫӢпјҡжҹҗе·Ҙе…·ж”ҜжҢҒ .ai/skills/
mkdir -p /path/to/your-project/.ai/skills
ln -s "$(pwd)/rust-style" /path/to/your-project/.ai/skills/rust-style

# жҲ–зӣҙжҺҘеӨҚеҲ¶
cp -r github-flow /path/to/your-project/.ai/skills/
```

еҰӮжһңе·Ҙе…·зәҰе®ҡдәҶзү№е®ҡзӣ®еҪ•пјҲеҰӮ `.cursor/skills/`пјүпјҢжҠҠдёҠйқўзҡ„ `.ai/skills/` жҚўжҲҗеҜ№еә”зӣ®еҪ•еҚіеҸҜгҖӮ

### дёӘдәәе…ЁеұҖ Skills

й»ҳи®Ө `make install` дјҡжҠҠ skill й“ҫжҺҘеҲ° Cursor е…је®№зӣ®еҪ• `~/.cursor/skills/`гҖӮTrae жңүдё“з”Ёзӣ®ж Үпјӣе…¶д»–е·Ҙе…·еҸҜйҖҡиҝҮ `SKILLS_DIR` жҢҮе®ҡзӣ®ж Үзӣ®еҪ•пјҡ

```bash
# CursorпјҲй»ҳи®Өпјү
make install

# Trae еӣҪйҷ…зүҲпјҡ~/.trae/skills/
make install-trae

# Trae еӣҪеҢәпјҡ~/.trae-cn/skills/
make install-trae-cn

# е®үиЈ…еҲ°иҮӘе®ҡд№ү skills зӣ®еҪ•
make install SKILLS_DIR=/path/to/your-ai-tool/skills
```

жҹҘзңӢй“ҫжҺҘзҠ¶жҖҒпјҡ

```bash
make list
make list-trae
make list-trae-cn
```

ж ЎйӘҢ skill е…ғж•°жҚ®дёҺ READMEпјҡ

```bash
make help
make check
```

еҚёиҪҪжң¬д»“еә“еҲӣе»әзҡ„й“ҫжҺҘпјҡ

```bash
make uninstall
make uninstall-trae
make uninstall-trae-cn

# еҚёиҪҪиҮӘе®ҡд№үзӣ®ж Үзӣ®еҪ•дёӯзҡ„й“ҫжҺҘ
make uninstall SKILLS_DIR=/path/to/your-ai-tool/skills
```

> еҰӮжһңдҪҝз”Ё CursorпјҢдёҚиҰҒж”ҫеҲ° `~/.cursor/skills-cursor/`пјҢйӮЈжҳҜ Cursor еҶ…зҪ® skill зӣ®еҪ•гҖӮ
> Trae йЎ№зӣ®зә§ skills зӣ®еҪ•жҳҜ `.trae/skills/`пјӣдёҠйқўзҡ„ `install-trae*` е®үиЈ…зҡ„жҳҜдёӘдәәе…ЁеұҖ skillsгҖӮ

### е…¶д»– AI зј–зЁӢе·Ҙе…·зӨәдҫӢ

**Claude Code / ж”ҜжҢҒ `CLAUDE.md` зҡ„е·Ҙе…·**

еңЁйЎ№зӣ® `CLAUDE.md` дёӯеҶҷпјҡ

```markdown
ејҖеҸ‘д»»еҠЎдјҳе…ҲеҸӮиҖғпјҡ
- /path/to/skills/development-workflow/SKILL.md
- /path/to/skills/clean-code/SKILL.md
- /path/to/skills/testing/SKILL.md

жҢүд»»еҠЎзұ»еһӢеҶҚиҜ»еҸ–еҜ№еә”иҜӯиЁҖжҲ–йўҶеҹҹ skillгҖӮ
```

**Codex / йҖҡз”Ё CLI Agent**

еңЁжҸҗзӨәиҜҚдёӯеҶҷпјҡ

```text
иҜ·жҢү /path/to/skills/development-workflow/SKILL.md еҲҶжһҗд»»еҠЎпјҢ
ж¶үеҸҠ Python ж—¶еҶҚеҸӮиҖғ /path/to/skills/python-style/SKILL.mdгҖӮ
```

**GitHub Copilot Chat / IDE и§„еҲҷ**

жҠҠеёёз”Ё skill зҡ„иҰҒзӮ№еӨҚеҲ¶еҲ°йЎ№зӣ®и§„еҲҷгҖҒд»“еә“иҜҙжҳҺжҲ– `AGENTS.md`пјҢе№¶дҝқз•ҷеҲ°жң¬д»“еә“ `SKILL.md` зҡ„й“ҫжҺҘпјҢйҒҝе…ҚеӨҡеӨ„и§„еҲҷжјӮз§»гҖӮ

### еңЁеҜ№иҜқдёӯдҪҝз”Ё

- ж”ҜжҢҒиҮӘеҠЁеҸ‘зҺ°зҡ„е·Ҙе…·дјҡж №жҚ® `SKILL.md` зҡ„ `description` еҢ№й…Қзӣёе…і skill
- д№ҹеҸҜеңЁеҜ№иҜқйҮҢзӣҙжҺҘзӮ№еҗҚпјҢдҫӢеҰӮпјҡгҖҢжҢү rust-style еҶҷиҝҷж®өд»Јз ҒгҖҚ

## ж–°еўһ Skill

1. ж–°е»әзӣ®еҪ•пјҡ`mkdir skill-name`
2. еҲӣе»ә `SKILL.md`пјҢеҢ…еҗ« frontmatterпјҡ

```markdown
---
name: skill-name
description: дёҖеҸҘиҜқиҜҙжҳҺеҒҡд»Җд№ҲгҖҒдҪ•ж—¶дҪҝз”ЁпјҲAgent йқ иҝҷж®өеҶіе®ҡжҳҜеҗҰеҗҜз”Ёпјү
---

# Skill Name

е…·дҪ“жҢҮд»ӨвҖҰвҖҰ
```

3. `description` еҶҷжё…жҘҡи§ҰеҸ‘жқЎд»¶пјӣжӯЈж–ҮеҸӘеҶҷ Agent жү§иЎҢж—¶йңҖиҰҒзҡ„и§„еҲҷпјҢйҒҝе…Қз©әиҜқ
4. иҝҗиЎҢ `make catalog` жӣҙж–° README skill зӣ®еҪ•
5. иҝҗиЎҢ `make check`

## з»ҙжҠӨжөҒзЁӢ

дҝ®ж”№жҲ–ж–°еўһ skill ж—¶пјҡ

1. дјҳе…Ҳдҝ®ж”№е·Іжңү skillпјӣеҸӘжңүиҒҢиҙЈдёҚеҗҢжҲ–и§ҰеҸ‘еңәжҷҜжҳҺжҳҫдёҚеҗҢжүҚж–°еўһ
2. `SKILL.md` дҝқжҢҒзІҫз®ҖпјҢеҺҹеҲҷдёҠе°‘дәҺ 500 иЎҢпјӣй•ҝз»ҶиҠӮж”ҫдёҖеұӮ `reference.md` / `examples.md`
3. е·Ҙе…·й“ҫе»әи®®еҢәеҲҶгҖҢж–°йЎ№зӣ®ејәзғҲжҺЁиҚҗгҖҚе’ҢгҖҢж—ўжңүйЎ№зӣ®и·ҹйҡҸгҖҚ
4. дёҚеҶҷзңҹе®һжң¬жңәи·Ҝеҫ„гҖҒеҜҶй’ҘгҖҒиҙҰеҸ·дҝЎжҒҜжҲ–е…¬еҸёеҶ…йғЁең°еқҖ
5. з”ЁжҲ·еҸҜи§ҒеҸҳжӣҙеҗҢжӯҘ `CHANGELOG.md`
6. дҝ®ж”№еҗҺиҝҗиЎҢпјҡ

```bash
make catalog
make check
make install
```

## зәҰе®ҡ

- зӣ®еҪ•еҗҚдҪҝз”Ё `kebab-case`
- `name` дёҺзӣ®еҪ•еҗҚдҝқжҢҒдёҖиҮҙ
- дёҖдёӘ skill еҸӘеҒҡдёҖд»¶дәӢпјӣи§„иҢғзұ»дёҺжөҒзЁӢзұ»еҲҶејҖ
- дјҳе…ҲеҶҷгҖҢеҝ…йЎ» / зҰҒжӯўгҖҚи§„еҲҷпјҢе°‘еҶҷиғҢжҷҜд»Ӣз»Қ
